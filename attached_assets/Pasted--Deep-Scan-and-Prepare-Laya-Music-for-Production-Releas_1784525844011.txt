# GitHub Actions

Create a complete GitHub Actions workflow inside:

```
.github/workflows/
```

The workflow must:

* Trigger automatically on:

  * push
  * pull_request
  * workflow_dispatch
  * release
  * version tags

The workflow should:

1. Checkout the repository
2. Set up Java
3. Set up Gradle
4. Cache Gradle dependencies
5. Build the Release APK
6. Verify the build succeeds
7. Upload the generated APK as a workflow artifact
8. Automatically create a GitHub Release when a version tag is pushed
9. Upload the generated APK to the GitHub Release

---

# Release APK

Generate a release APK named exactly:

laya.apk

The workflow should automatically rename the generated APK to:

laya.apk
